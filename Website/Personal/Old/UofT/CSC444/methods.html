METHODS

Our group is fully affiliated with CMI.


This implies that all of our code obides by the strict outline given in the standard.

Therefore, our method names, arguments, return types, etc. all conform to the well organized specifications constructed by Macrosoft.

Specifications may be accessed through the CMI Documentation.
Here is a quick glance at our Method prototypes:

note: ***For Most Up To Date Specifications, Consult CMI Documentation***

Method Prototype Description
void addMoveableObj (UniqueId roomId, char type, UniqueId objectId) Given the unique id of a moveable object, its class (creature or artifact) and the unique id of a room, adds the object to the relevant list of things in that room.
void appendDescription (UniqueId roomId, java.lang.String text) Given the unique id of a room, and a piece of text, appends the text to the description of the room.
java.util.List getArtifactList (UniqueId roomId) Given the unique id of a room, provides a list of unique ids of all artifacts in that room.
java.util.List getCreatureList (UniqueId roomId) Given the unique id of a room, provides a list of unique ids of all creatures in the room.
java.lang.String getDescription (UniqueId roomId) Given the unique id of a room, returns a textual description of that room.
java.util.List getPortalList (UniqueId roomId) Given the unique id of a room, provides a list containting the direction and description of every portal in the room.
UniqueId getSuccessor (UniqueId roomId, char direction) Given the unique id of a room, and a direction, returns the unique id of the next adjacent room in that direction.
java.util.List initialize (java.util.List ids) Given a list of unique ids of moveable objects, randomly places the objects in various rooms, and returns an ordered list of the unique ids of the rooms the moveable objects were placed in.
void loadMap() Loads the map from a configuration file.
void removeMoveableObj (UniqueId roomId, char type, UniqueId objectId) Given the unique id of a moveable object, its class (creature or artifact) and the unique id of a room, removes the object from the relevant list of things in that room.
void setAM( IArtifactManager amRef) Sets reference to Artifact Manager.
void setCFM (ICreatureAndFightManager cfmRef) Sets reference to Creature and Fight Manager.
note: ***For Most Up To Date Specifications, Consult CMI Documentation***
